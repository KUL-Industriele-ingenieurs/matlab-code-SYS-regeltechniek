# Matlab code for SYS-regeltechniek

This repository contains Matlab exercises and examples for the course **Systeemtheorie en regeltechniek** (Industrial Engineering EM 2025-2026).

## How to get this code (for beginners)

If you are not familiar with Git, the easiest way to get these files is to download them as a ZIP folder:

1.  Go to the main page of this repository: [https://github.com/KUL-Industriele-ingenieurs/matlab-code-SYS-regeltechniek](https://github.com/KUL-Industriele-ingenieurs/matlab-code-SYS-regeltechniek)
2.  Click the green **Code** button near the top right.
3.  Select **Download ZIP** from the dropdown menu.
4.  Once downloaded, extract the ZIP file on your computer.
5.  Open the folder in Matlab to run the scripts.

---

### For Git users
If you have Git installed, you can clone this repository using:

git clone https://github.com/KUL-Industriele-ingenieurs/matlab-code-SYS-regeltechniek.git
